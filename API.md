### API Reference

###### Array

**head** : ```js _.head(array)```

Gets the first element of an array

```js
_.head([1, 2, 3]) // 1
_.head([]) // undefined
```

**_.tail(array)**

Gets all but the first element of an array.

```js
_.tail([1, 2, 3]) // [2, 3]
_.tail([]) // []
```

**_.last(array)**

Gets the last element of an array.

```js
_.last([1, 2, 3]) // 3
_.last([]) // undefined
```
