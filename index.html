pareto.js

A small trully functional utility library that only has what you really need.

View on GitHub Download .zip Download .tar.gz

Advantages

  1. Lightweight
  2. Functional
    • Only pure functions
    • No side-effects
    • Encourages immutability
    • Auto-curried methods
  3. Better and smaller API
  4. Made with the newest stuff:
    • ES6
    • Babel
    • Webpack

Installation

To install the stable version:

npm install --save paretojs

This assumes that youâ€™re using npm with a module bundler like Webpack

How

Import pareto.js:

import _ from 'paretojs'

And use it:

const myArray = [1, 2, 3]
const newArray = _.remove(myArray, 2)

console.log(newArray) // [1, 2]
